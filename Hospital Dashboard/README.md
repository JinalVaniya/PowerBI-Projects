## Project Description

This project is a Hospital Emergency Room Dashboard created in Power BI to analyze and visualize ER data in a simple and interactive way. Before building the dashboard, the data was cleaned, processed, and filtered using Power Query to remove inconsistencies and prepare it for analysis. DAX was used to create the required measures and KPIs.

The dashboard provides insights into important metrics such as the total number of patients, average wait time, patient satisfaction score, admission status, referrals, patient demographics, and patient visits by day and hour. It also includes a month and year filter so users can easily explore data for different time periods. This dashboard helps present hospital data in a clear and meaningful way, making it easier to understand trends and support better decision-making.
