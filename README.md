## Project Description

This project is an interactive HR Analytics Dashboard built in Power BI to analyze employee attrition and workforce trends. Before creating the dashboard, the data was cleaned, processed, and transformed using Power Query to improve data quality and consistency. DAX was used to create KPIs and calculated measures for analysis.

The dashboard provides insights into employee count, attrition rate, average age, average salary, years at the company, attrition by department, job role, age group, education, salary slab, and gender. It helps HR teams identify attrition patterns and supports better workforce planning and decision-making.

## Project Description

This project is an interactive Power BI dashboard created using survey data from data professionals across different countries. Before building the dashboard, the data was cleaned, processed, and transformed using Power Query to ensure accurate analysis. DAX was used to create measures and calculations for the visualizations.

The dashboard provides insights into average salary by job role and gender, favorite programming languages, work-life balance, salary satisfaction, survey participants by country, and demographic information such as age. It helps explore trends in the data and understand the preferences and experiences of professionals working in the data field.
